# ローマ字れんしゅう

ひらがなからローマ字をおぼえる、小学生むけの練習サイトです。
ブラウザだけで動きます。サーバーもビルドも不要です。

## 中身

| ファイル | 内容 |
|---|---|
| `index.html` | トップページ（メニュー） |
| `moji.html` | **もじ編** ひらがな1文字を見て、正しいローマ字を4択でえらぶ |
| `kotoba.html` | **ことば編** ことばを見て、文字タイルをならべて書く |

3つとも1ファイル完結です。画像もフォント以外の外部ファイルもありません。

## できること

- **あ行から順にすすむ**（ことば編）
  あ行 → か行 → … → だくおん → つまる音 → ようおん の13段階。
  選んだ行までの文字だけで書けることばが出題されます。

  | 選んだ範囲 | 出ることばの数 |
  |---|---|
  | あ行まで | 15語 |
  | さ行まで | 80語 |
  | ら行まで | 316語 |
  | ようおんまで | 594語 |

- **出ることばを1語ずつえらべる**（ことば編）
  行ごとの一覧からタップで選択。ひらがなを入力すれば自分でことばを追加でき、
  ローマ字は自動で作られます。

- **書き方をえらべる**
  ヘボン式（shi・tsu・sha）／ 訓令式（si・tu・sya）

- **アルファベットをえらべる**
  小文字 abc ／ 大文字 ABC

- **そのほか**
  読み上げ（対応ブラウザ）、まちがえたことばの復習、正解ではなまる。

## ローマ字の書き方について

- つまる音（っ）は次の子音を重ねます … がっこう → `gakkou`
- ようおんは ヘボン式 `sha` / 訓令式 `sya` … しゃしん → `shashin`
- のばす音はそのまま書きます … おとうさん → `otousan`
  （`ō` のような記号は使っていません。キーボードで打つときと同じ書き方です）
- 「ん」は `n` です … せんせい → `sensei`

## 公開のしかた（GitHub Pages）

1. このリポジトリの **Settings** → 左メニューの **Pages**
2. Source を **Deploy from a branch**、Branch を **main** / **/(root)** にして **Save**
3. 1〜2分で `https://ユーザー名.github.io/リポジトリ名/` で公開されます

## 動作環境

Chrome・Safari・Edge・Firefox の新しめのバージョン。スマホ・タブレットにも対応しています。
読み上げは端末に日本語の音声が入っている場合のみ動きます。
